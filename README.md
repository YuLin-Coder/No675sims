## 项目介绍
基于SSM框架的工资信息管理系统，有员工和管理员两个角色，主要功能如下
用户：月度工资、年度工资、修改个人信息、消息中心、工资图标统计等
管理员：添加员工信息、部门管理、管理员工信息、发布工资信息、修改账户密码等

## 项目技术
- 编程语言：Java
- 数据库：MySQL
- 前端技术：JSP、Layui、Jquery
- 后端技术：Spring、SpringMVC、MyBatis

## 运行环境
- JDK版本：JDK1.8及以上
- 开发工具：IDEA、Ecplise、Myecplise都可以
- 数据库: MySQL5.7及以上
- Maven：maven3.0及以上
